## ✨ Tính năng Nổi bật

### 🛒 Đối với Khách hàng
*   **Giao diện Hiện đại:** Thiết kế chuyên nghiệp, responsive (tương thích tốt trên điện thoại và máy tính).
*   **Danh mục Sản phẩm:** Phân loại rõ ràng (Thịt Bò, Thịt Heo) với bộ lọc giá và sắp xếp thông minh.
*   **Giỏ hàng & Thanh toán:** Quy trình mua hàng tối ưu, lưu trữ giỏ hàng tại chỗ (Local Storage).
*   **Tra cứu Đơn hàng:** Khách hàng có thể dễ dàng kiểm tra tình trạng đơn hàng của mình.
*   **Hỗ trợ Đa kênh:** Tích hợp nút Chat Zalo, Hotline và Chat trực tiếp.

### 🛠️ Đối với Quản trị viên (Admin)
*   **Quản lý Sản phẩm:** Thêm mới, chỉnh sửa thông tin, cập nhật giá nhanh (inline-edit).
*   **Quản lý Kho hàng:** Bật/Tắt hiển thị sản phẩm, cập nhật trạng thái "Còn hàng" hoặc "Hết hàng".
*   **Quản lý Đơn hàng:** Hệ thống lọc đơn hàng theo trạng thái (Chờ xác nhận, Đang chuẩn bị, Đang giao, Hoàn tất...).
*   **Thống kê Khách hàng:** Theo dõi danh sách khách hàng và tổng mức chi tiêu.

## 🛠️ Công nghệ Sử dụng

Dự án được xây dựng với các công nghệ web hiện đại mà không cần framework phức tạp, đảm bảo tốc độ tải trang cực nhanh:

*   **Cơ bản:** HTML5, CSS3, JavaScript (ES6+).
*   **Giao diện:** 
    *   **Google Fonts:** Outfit (font chữ hiện đại, cao cấp).
    *   **FontAwesome 6.4:** Hệ thống icon trực quan.
*   **Logic:** Xử lý dữ liệu động và lưu trữ trạng thái qua `localStorage`.
*   **Thông báo:** Hệ thống Custom Notification (Toast) tích hợp sẵn.
